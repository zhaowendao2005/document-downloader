Helicone
本页介绍如何在LangChain中使用
Helicone
。
Helicone 是什么？
​
Helicone是一个开源的观测平台，代理您的OpenAI流量，并为您提供有关您的花费、延迟和使用情况的关键见解。
快速入门
​
在您的LangChain环境中，您只需添加以下参数。
const
model
=
new
OpenAI
(
{
}
,
{
basePath
:
"https://oai.hconeai.com/v1"
,
}
)
;
const
res
=
await
model
.
call
(
"What is a helicone?"
)
;
现在，前往
helicone.ai
创建您的帐户，并在我们的仪表板中添加OpenAI API密钥以查看日志。
