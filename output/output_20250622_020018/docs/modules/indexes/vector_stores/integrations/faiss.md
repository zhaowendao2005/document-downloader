Faiss
兼容性
仅适用于Node.js环境。
Faiss
是用于稠密向量的高效相似度搜索和聚类的库。
Langchainjs支持使用Faiss作为向量库，并可将其保存到文件。同时，它还提供从
Python实现
读取保存的文件的功能。
安装
​
安装
faiss-node
,它是
Faiss
的Node.js绑定。
npm
Yarn
pnpm
npm
install
-S faiss-node
yarn
add
faiss-node
pnpm
add
faiss-node
要启用从
Python实现
读取保存的文件的功能，还需要安装
pickleparser
。
npm
Yarn
pnpm
npm

-S pickleparser
yarn
add
pickleparser
