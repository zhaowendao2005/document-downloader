兼容性
仅限于 Node.js。
OpenSearch
是完全兼容 Elasticsearch API 的 Elasticsearch 分支。在此处阅读有关其支持近似最近邻的更多信息
here
。
Langchain.js 将
@opensearch-project/opensearch
作为 OpenSearch vectorstore 的客户端。

npm
Yarn
pnpm

yarn

@opensearch-project/opensearch
pnpm

您还需要运行一个 OpenSearch 实例。您可以使用

此处
