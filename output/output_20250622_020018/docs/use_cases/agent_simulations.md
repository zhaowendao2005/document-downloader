代理人模拟
代理人模拟涉及到将多个代理人与彼此互动。
它们往往使用模拟环境，其中LLM是它们的“核心”，而辅助类则用于提示它们吸收某些输入，例如预先构建的“观察”，并对新的刺激作出反应。
它们还从长期记忆中受益，以便它们可以在互动之间保持状态。
